An react demo page 
## Prerequisites

* NodeJS / NPM
* Gulp

```
npm install -g gulp
```

## Development instructions

### Install Dependencies

```
npm install
```

### Run Dev Environment

```
gulp
```

### Turn off first run rebuilds and endless lifereload updates on jade change:

```
gulp --light
```

<!-- ### Test

```
gulp test
``` -->

### Prepare Assets for Deployment

```
gulp deploy
```
